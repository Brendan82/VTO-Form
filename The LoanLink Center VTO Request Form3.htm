The LoanLink Center VTO Request Form
Please fill out this form completely by entering your information in all of the following fields:
1.  Select the name of your Primary Campaign

2.  Enter the date for which you are requesting to VTO

3.  Enter your desired time to leave

4.  Select whether this will be Paid or Unpaid
     (Paid can only be used when PTO is available)
Note:  Any information not included will serve to delay VTO request approval.  Please only send one VTO per requested day.  There is no limit to the number of VTO requests you can have, but you can only request VTO once for a given day.  If your request is approved, you will receive a pop up from IEX.
